"Slightly Magic" Available for Pre-Order

22 June 2014 15:17 NZST

The remake of “Slightly Magic” by Colin Jones is almost ready for release and can now be pre-ordered. Earlier this year Colin ran a sadly unsuccessful Kickstarter for it, so it’s great to see this will indeed see the light of day. The game features the original’s graphics and a remastered soundtrack by Allister Brimble.
